# CPP-Submissions

A growing collection of C++ practice — DSA problems, algorithm implementations, design pattern examples, and concept notes. Organized by topic for easy navigation.

## 📁 Structure

- **DataStructures/** — Linked lists, stacks, queues, trees, graphs, and other core data structures
- **Algorithms/** — Sorting, binary search, dynamic programming, recursion, and other core algorithms
- **LeetCode-Problems/** — Solutions to LeetCode-style interview problems
- **DesignPatterns/** — SOLID principles and other OOP design pattern implementations
- **Concepts-Notes/** — Reference notes on C++, OS fundamentals, graphics APIs, and more
- **Misc/** — Uncategorized files pending review

## 🛠️ Usage

Each `.cpp` file is a standalone solution and can be compiled independently:

```bash
g++ FileName.cpp -o FileName
./FileName
```

## 🎯 Purpose

This repo serves as a personal practice log and reference — covering fundamental data structures, common interview-style problems, and design pattern implementations in C++. Updated regularly as new problems are solved and new notes are added.
